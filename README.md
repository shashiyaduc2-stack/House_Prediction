# House_Prediction
#  House Price Prediction & Analysis using Machine Learning

##  Project Overview

This project focuses on analyzing housing data and building multiple machine learning models for **classification, regression, and clustering**. The goal is to extract meaningful insights, predict house prices, and group similar houses.

---

##  Objectives

* Perform Exploratory Data Analysis (EDA)
* Visualize patterns and relationships in the dataset
* Apply preprocessing techniques (encoding, scaling)
* Build classification and regression models
* Apply clustering for grouping houses
* Evaluate and compare model performance

---

##  Dataset Description

The dataset contains housing-related features such as:

* Longitude & Latitude
* Housing Median Age
* Total Rooms & Bedrooms
* Population & Households
* Median Income
* Ocean Proximity (Categorical Feature)
* Median House Value

---

##  Exploratory Data Analysis (EDA)

* Checked dataset shape and structure
* Identified missing values
* Performed statistical analysis
* Observed relationships between features

---

##  Data Visualization

* Distribution plots
* Correlation heatmap
* Scatter plots (Income vs House Value)
* Count plots for categorical variables

---

##  Data Preprocessing

* Handled missing values using mean imputation
* Applied **One-Hot Encoding** for categorical variables
* Used **StandardScaler** for feature scaling
* Removed unnecessary features

---

##  Machine Learning Models

### Classification Models

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier

 Used to classify houses into categories (e.g., price ranges)

---

### 🔹 Regression Models

* Linear Regression
* Lasso Regression
* Decision Tree Regressor
* Random Forest Regressor
* Gradient Boosting Regressor
* KNN Regressor
* Support Vector Regressor (SVR)

 Used to predict house prices

---

## Model Evaluation

### For Classification:

* Accuracy Score
* Confusion Matrix

### For Regression:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

---

##  Clustering Analysis

* Applied **K-Means Clustering**
* Used **Elbow Method** to determine optimal clusters
* Grouped houses into categories (low, medium, high value)

---

##  Key Insights

* Median income strongly influences house prices
* Location and proximity to ocean impact pricing
* Ensemble models like Random Forest give better performance
* Clustering helps identify different housing segments
